Execute in following command in cmd to compile:
gcc /path/to/main.c -std=c99 -o <output_file_name>
Run the program by:
path/to/<output_file_name> <usage_message>

<usage_message>
1, q6, int a, int b, int c
Randomly generate and print two axb and bxc matrices, multiply them and then
print the result.

2, q8, int min, int max, int step, int replication
print time to multiply two NxN matrices
the N starts from min, end at max and vary with step
the average time each N corresponds is the mean time of replication
